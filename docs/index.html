Create TanStack Boilerplate

A fast, production-ready CLI to scaffold TanStack Start applications (React + Vite + TypeScript + Tailwind). Generate an opinionated starter with optional i18n, auth, UI components, and testing.

View on npm • View on GitHub

Quick Install

npx create-tanstack-boilerplate

Why use this?

  • Opinionated defaults for fast bootstrapping of TanStack Start apps.
  • Interactive CLI to pick features: i18n, UI, auth, testing.
  • Built for production: Vite, TypeScript, Tailwind integration.
